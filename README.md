# Health & Wellness Data Science Project

## Overview
A data science project analyzing how hydration, sleep, and BMI impact daily productivity using Python and Machine Learning.

Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- HTML, CSS, JavaScript

Project Components
 1. Health Assistant Dashboard
- Tracks daily water intake, BMI, and schedules
- Built using HTML, CSS, and JavaScript

2. Data Analysis & Prediction
- Data cleaning and EDA
- Correlation analysis
- Linear Regression model (R² ≈ 0.85)

Key Insights
- Positive correlation between hydration, sleep, and productivity
- Demonstrates full ML workflow

How to Run
1. Open `index.html` in browser
2. Run `wellness_analysis.ipynb` in Jupyter Notebook
